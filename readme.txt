== Description ==
This WordPress Theme “Material Food” has been developed by the students of the second year of Scuola Mohole, in Milano, Italy, during a workshop, March 2016, in collaboration of the tutor Francesco Riggio.
The goals of the theme was "style, innovation, functionality", using open-source techniques as Flexbox, Material Design Lite, Less.

For a demo and more information about "Material Food" please go to http://www.mohole.it/web/food

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Material Food in the search form and press the 'Enter' key in your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel.
